Method. Each licence is assigned to a district using the first two digits of its 6-digit postal code (the official URA postal-sector→district table). The map's shapes are an approximate district boundary built from URA planning areas and are shown for visual context only. Counts. 30,558 licences are mapped to districts. A further 6,083 licences are not shown because they have no postal code in the address (6,073) or an offshore/special sector such as 90 Pulau Bukom (10). Source: NEA List of Licensed Eating Establishments.
